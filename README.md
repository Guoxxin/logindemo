# logindemo
## 一个基于spring boot+vue.js的登录demo
### 文件介绍
* webApp -- vue文件
* logindemo -- spring boot文件
* vueblog.sql --数据库文件
### 使用流程
1. 在webstorm 中导入webApp文件；
2. 在IDEA中导入logindemo文件；
3. 在Mysql中执行vueblog.sql文件；
4. 在application.yml 中修改自己的数据用户名和密码；
5. 启动前后台项目及mysql服务；
6. 访问localhost：8080 输入用户名+密码即可体验登录
